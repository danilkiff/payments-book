褋褌芯泄泻芯褋褌懈 写谢褟 薪芯胁褘褏 褋懈褋褌械屑.

芯褌~褋褌芯褉芯薪褘, 胁谢邪写械褞褖械泄 泻谢褞褔芯屑. 袙~ISO~8583 MAC 褉邪蟹屑械褖邪褞褌 胁~DE~64

    Integrity (褑械谢芯褋褌薪芯褋褌褜 泻芯屑邪薪写 褝屑懈褌械薪褌邪 泻~褔懈锌褍, issuer~scripts),

    蟹邪锌懈褋褘胁邪械褌褋褟 胁~褔懈锌 锌褉懈 锌械褉褋芯薪邪谢懈蟹邪褑懈懈.

褍褋褌褉芯泄褋褌胁, 邪~懈蟹 薪懈褏 锌芯~薪芯屑械褉褍 褌褉邪薪蟹邪泻褑懈懈 胁褘胁芯写褟褌褋褟 褉邪斜芯褔懈械 泻谢褞褔懈

Initial Key (IK) 芯写薪懈屑 斜谢芯泻芯屑 AES 胁~褉械卸懈屑械 ECB:
$$\text{IK} = \text{AES-ECB}_{\text{BDK}}(\text{DD}_{\text{init}}),$$

写谢褟 Initial Key, \textit{algorithm} = AES-128, \textit{length} = 128) 懈~屑谢邪写褕懈屑懈
8~斜邪泄褌邪屑懈, 褉邪胁薪褘屑懈 IKI~\cite{ansi-x9-24-3}. 孝械褉屑懈薪邪谢 锌芯谢褍褔邪械褌 IK
懈~褍写邪谢褟械褌 谢褞斜褘械 褋谢械写褘 BDK. 小~褝褌芯谐芯 屑芯屑械薪褌邪 BDK 褋褍褖械褋褌胁褍械褌 褌芯谢褜泻芯 胁~HSM.

懈薪泻褉械屑械薪褌懈褉褍械褌褋褟, 薪邪斜芯褉 IDK 锌械褉械褋褌褉邪懈胁邪械褌褋褟 锌芯写 薪芯胁褘械 褍褋褌邪薪芯胁谢械薪薪褘械
斜懈褌褘, 邪~懈褋锌芯谢褜蟹芯胁邪薪薪褘泄 working key 褋褌懈褉邪械褌褋褟. 小械褉胁械褉薪邪褟 褋褌芯褉芯薪邪
(褏芯褋褌 褝泻胁邪泄械褉邪 懈谢懈 锌谢邪褌褢卸薪邪褟 褋械褌褜), 蟹薪邪褟 BDK 懈~锌芯谢褍褔械薪薪褘泄 褋~褌褉邪薪蟹邪泻褑懈械泄
