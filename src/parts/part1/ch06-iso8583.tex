袚褉邪屑屑邪褌懈泻邪 ISO~8583 芯写懈薪邪泻芯胁邪 写谢褟~胁褋械褏 褋芯芯斜褖械薪懈泄 懈~褋芯褋褌芯懈褌 懈蟹~褌褉褢褏 褔邪褋褌械泄, 懈写褍褖懈褏 褋褌褉芯谐芯 胁~芯写薪芯屑 锌芯褉褟写泻械:

\begin{enumerate}
  \item \textbf{MTI} (Message Type Indicator)~--- 褔械褌褘褉械 锌械褉胁褘褏 斜邪泄褌邪 (\texttt{30 31 30 30}, 胁~ASCII 写邪褢褌~\texttt{0100}). 袨褌胁械褔邪褞褌 薪邪~胁芯锌褉芯褋, \emph{褔褌芯 褝褌芯 蟹邪~褋芯芯斜褖械薪懈械}: 邪胁褌芯褉懈蟹邪褑懈褟, 褎懈薪邪薪褋芯胁芯械 褋芯芯斜褖械薪懈械, 芯褌屑械薪邪, 褋械褌械胁芯械 褍锌褉邪胁谢械薪懈械.
  \item \textbf{Bitmap}~--- 褋谢械写褍褞褖懈械 胁芯褋械屑褜 斜邪泄褌 (\texttt{72 3C 04 81 20 C0 80 00}). 袘懈褌芯胁邪褟 泻邪褉褌邪, 芯褌胁械褔邪褞褖邪褟 薪邪~胁芯锌褉芯褋, \emph{泻邪泻懈械 锌芯谢褟 锌褉懈褋褍褌褋褌胁褍褞褌 写邪谢褜褕械}.
  \item \textbf{协谢械屑械薪褌褘 写邪薪薪褘褏} (data elements, DE)~--- 褋邪屑懈 蟹薪邪褔械薪懈褟: 胁~薪邪褕械屑 褋芯芯斜褖械薪懈懈 锌褟褌薪邪写褑邪褌褜 锌芯谢械泄: PAN (Primary Account Number; 褉邪蟹斜芯褉~--- 胁~谐谢.~\ref{ch:tx-lifecycle}), 褋褍屑屑邪, STAN (Systems Trace Audit Number, 薪芯屑械褉 褌褉邪褋褋懈褉芯胁泻懈), 泻芯写 芯斜褉邪斜芯褌泻懈, 懈写械薪褌懈褎懈泻邪褌芯褉褘 褌械褉屑懈薪邪谢邪 懈~孝小袩, 胁褉械屑械薪薪褘械 屑械褌泻懈 懈~械褖褢 薪械褋泻芯谢褜泻芯 锌褉懈蟹薪邪泻芯胁 褌褉邪薪蟹邪泻褑懈懈.
\end{enumerate}

\highlight{ISO~8583 褍褋褌褉芯械薪 懈薪邪褔械, 褔械屑 褋邪屑芯芯锌懈褋褘胁邪褞褖懈械褋褟 褎芯褉屑邪褌褘 胁褉芯写械 JSON 懈谢懈~XML. 小薪邪褔邪谢邪 懈写褢褌 褌懈锌 芯锌械褉邪褑懈懈, 蟹邪褌械屑 bitmap 褋~泻邪褉褌芯泄 锌褉懈褋褍褌褋褌胁懈褟, 蟹邪褌械屑 蟹薪邪褔械薪懈褟 锌芯写褉褟写 胁~锌芯褉褟写泻械 胁芯蟹褉邪褋褌邪薪懈褟 薪芯屑械褉芯胁}. 孝械谐 懈~写谢懈薪邪 胁薪褍褌褉懈 泻邪卸写芯谐芯 锌芯谢褟 薪械~锌械褉械写邪褞褌褋褟.
