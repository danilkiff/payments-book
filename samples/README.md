# Python-примеры из книги

Каждый файл `*.py` в `samples/<глава>-<тема>/` — ровно тот код, который
попадает в книгу через `\codefile`. Тесты в `test_*.py` верифицируют
поведение, заявленное в прозе.

```bash
make            # справка по командам
make sync       # установить .venv (требуется uv)
make check      # формат + линт + тесты
```
