# Инженерия платежей

Исходники книги о карточных платежах, СБП и платёжной инфраструктуре.

```bash
# macOS
brew install mactex tex-fmt librsvg
make pdf

# Ubuntu
sudo apt install texlive-full tex-fmt librsvg2-bin  # tex-fmt: Ubuntu 25.10+
make pdf
```
